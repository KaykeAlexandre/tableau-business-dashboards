# Tableau Business Dashboards

## Visão Geral
Coleção de dashboards desenvolvidos no Tableau com foco em análise de negócios, visualização de dados e geração de insights estratégicos.

Os projetos abordam diferentes cenários analíticos, incluindo vendas, métricas de e-mail marketing e análise de sessões de usuários.
